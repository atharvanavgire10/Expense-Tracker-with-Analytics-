# 💰 Expense Tracker

A full-stack Expense Tracker web application built using the MERN stack that helps users efficiently manage their personal finances. Users can securely record income and expenses, categorize transactions, monitor spending patterns, and visualize financial data through interactive charts and analytics.

---

## 🚀 Features

* 🔐 Secure User Authentication using JWT
* 💵 Add, Edit, and Delete Income & Expense Transactions
* 📊 Interactive Dashboard with Financial Analytics
* 📈 Visual Reports using Charts
* 📂 Transaction History Management
* 📱 Fully Responsive Design
* ⚡ RESTful API Architecture
* 🔒 Protected Routes and Authorization

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/expense-tracker.git
```

### 2. Navigate to the Project

```bash
cd expense-tracker
```

### 3. Install Backend Dependencies

```bash
cd backend
npm install
```

### 4. Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

## ▶️ Running the Application

### Start Backend Server

```bash
cd backend
npm run dev
```

### Start Frontend

```bash
cd frontend
npm run dev
```

Frontend:

```text
http://localhost:5173
```

Backend:

```text
http://localhost:8000
```

> Replace the backend port if your project uses a different one.

---

## 🔐 Environment Variables

Create a `.env` file inside the `backend` folder.

```env
PORT=8000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

CLIENT_URL=http://localhost:5173
```
